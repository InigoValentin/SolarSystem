Solar System
====

## A JavaScript simulation of the Solar System ###
<img src="http://i.imgur.com/UrvVDRA.png"/>

### Features ###

* Switch between the full Solar System or the Inner System.
* Tooltips with usefull info about the planets and the Sun.
* Elliptical orbits.
* Draw or hide orbits.
* Enlarge the planets or show the in their actual scale.
